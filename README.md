# 115云盘磁力链接助手

一个简单的油猴脚本，可以自动检测页面上的磁力链接并一键保存到115云盘。

## 功能特点

- 自动检测页面上的磁力链接
- 在磁力链接旁显示"115"按钮
- 一键保存磁力链接到115云盘
- 支持隐藏的磁力链接检测
- 实时页面监控，支持动态加载的内容
- 友好的视觉反馈和通知提示

## 安装

1. 首先安装油猴扩展 [Tampermonkey](https://www.tampermonkey.net/)
2. 点击 [这里](安装链接) 安装脚本
   - 或者打开油猴的管理面板
   - 创建新脚本
   - 复制本项目中的代码并保存

## 使用方法

1. 登录115云盘网页版
2. 访问任何包含磁力链接的网页
3. 脚本会自动在磁力链接旁显示"115"按钮
4. 点击按钮即可保存到115云盘

## 按钮状态说明

- 蓝色：正常状态
- 深蓝色：点击状态
- 红色：保存失败
- 悬停时会放大显示

## 注意事项

- 使用前需要先登录115云盘
- 部分网站可能会屏蔽磁力链接
- 请确保有足够的离线空间
- 遵守相关法律法规
